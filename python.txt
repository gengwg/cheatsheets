defaultdict means that if a key is not found in the dictionary, then instead of a KeyError being thrown, a new entry is created. The type of this new entry is given by the argument of defaultdict.

For example:

somedict = {}
print(somedict[3]) # KeyError

someddict = defaultdict(int)
print(someddict[3]) # print int(), thus 0

if you just want to return "bar": somedict = defaultdict(lambda:"bar")

# regular expression
^ matches the beginning of a string.
$ matches the end of a string.
\b matches a word boundary.
\d matches any numeric digit.
\D matches any non-numeric digit.
x? matches an optional x character (an x zero or one times).
x* matches x zero or more times.
x+ matches x one or more times.
x{n,m} matches an x character at least n times, but not more than m times.
(a|b|c) matches either a or b or c.
(x) is a remembered group. I can get the value of what matched by using the groups() method of the object returned by re.search().

### [[0]*n for _ in range(m)] VS [[0]*n]*m
        
>>> n = 3
>>> m = 3
>>> a = [[0]*n]*m
>>> a
[[0, 0, 0], [0, 0, 0], [0, 0, 0]]
>>> b = [[0] *n for _ in range(m)]
>>> b
[[0, 0, 0], [0, 0, 0], [0, 0, 0]]
>>> a == b
True

# for a it is a reference after *m
# while for b it is all copy!
>>> a[2][2] = 3
>>> a
[[0, 0, 3], [0, 0, 3], [0, 0, 3]]
>>> b[2][2] = 3
>>> b
[[0, 0, 0], [0, 0, 0], [0, 0, 3]]
