# tmux shortcuts & cheatsheet

// vim: tabstop=4 expandtab shiftwidth=4 softtabstop=4

start new session with name:

    tmux new -s myname

attach to named session:

    tmux a -t myname

detach a session:

    Ctrl + 'b', then press 'd'

list sessions:

    tmux ls

enter scroll mode:

    Ctrl+b [

quit scroll mode:

    q

create another window:

    Ctrl+b c

list windows:

    Ctrl+b w

go to window #:

    Ctrl+b #
