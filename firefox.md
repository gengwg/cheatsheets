### Disable accidental `ctrl+q`

1. Go to `about:config`
2. Set `browser.showQuitWarning = true`

### Add-ons

* HTTPS Everywhere 
* uBlock Origin
* Privacy Badger

### Shortcuts

`about:addons`

```
Ctrl + L    Go to address bar
Ctrl + K    Go to search bar
```

### Chinese Font (v57)
