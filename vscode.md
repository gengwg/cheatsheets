## Extensions

To install extensions, Go to 'Extensions' and search name.

- Markdown All In One
- Markdown Shortcuts
- Git Lens
- Git History


### preview markdown

Mac:

```
cmd + k, v
```

or separate tab:

```
cmd + shift+ v
```

Linux:

```
ctrl + shift+ v
```

### comment out lines

```
cmd + /
```

### Open current whole folder

```
code .
```
